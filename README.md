# Mintlify Code like how it is used and what is used where

## Where things are

### HTML (index.html)

- **navbar-wrapper** - top nav bar with logo and menu
- **logo-img** - the mintlify logo 
- **menu-list** - Resources, Documentation, Customers, Blog, Pricing links
- **sales-btn** - "Contact sales" button
- **free-btn** - "Start for free" button
- **showcase-area** - big hero section with title and form
- **highlight-box** - the green "NEW" badge
- **status-label** - the text "NEW" inside the badge
- **main-text** - the hero description text
- **form-container** - email input form
- **companies-workwith** - company logos section
- **intelligence-age** - features section (LLMs, agent, assistant)
- **enterprise-section** - enterprise stuff
- **customers-section** - customer stories with cards
- **cta-section** - final buttons (Get started, Get demo)
- **primary-btn** - "Get started for free" button
- **secondary-btn** - "Get a demo" button
- **site-footer** - footer with links

### CSS (styles.css)

- **.navbar-wrapper** - navigation bar styling
- **.logo-img** - logo width 100px
- **.menu-list** - nav links layout (flexbox)
- **.showcase-area** - hero section centered layout
- **.highlight-box** - green badge background color
- **.form-container** - teal email form styling
- **.primary-btn** - black button (Get started)
- **.secondary-btn** - white button with border (Get demo)
- **.customers-grid** - 4 column card grid
- **.site-footer** - footer styling

### Colors Used

- `--nav-text-color: #b4cad3` - light blue text
- `--suggestion-background-color: #044d6b` - dark blue backgrounds
- `#0c8c5e` - green accents
- `#71b0ab` - teal form color
